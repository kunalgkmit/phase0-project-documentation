# UI/UX Overview

## Screens
1. **Signup Screen** — User registration via JWT.  
2. **Login Screen** — User authentication.  
3. **Verify Email Screen** — Optional email verification.  
4. **Home Dashboard** — Displays balance summary and recent transactions.  
5. **Create Transaction Screen** — Add income or expense entries.  
6. **Charts Screen** — Visualize income vs expense by date range.  
7. **Admin Screen** — Displays total users and names.

## Design Principles
- Minimal and clean interface using React Native components.
- Consistent color scheme for income (green) and expense (red).
- Pull-to-refresh interaction for data reload.
