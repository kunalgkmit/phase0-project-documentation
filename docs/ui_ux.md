# UI/UX Overview

## Screens
1. **Signup Screen** — User registration via JWT.  
2. **Login Screen** — User authentication.  
3. **Home Dashboard** — Displays balance summaries and recent transactions.  
4. **Create Transaction Screen** — Add income or expense entries.  
5. **Admin Screen** — Displays total users their names.

## Design Principles
- Minimal and clean interface using Flutter.
- Consistent color scheme for income (green) and expense (red).
- Pull-to-refresh interaction for data reload.
