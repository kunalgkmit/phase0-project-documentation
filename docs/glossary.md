# Glossary

| Term | Definition |
|------|-------------|
| **CRUD** | Create, Read, Update, Delete operations for transactions. |
| **Admin** | User with elevated privileges to view registered users. |
| **Dashboard** | Main screen showing financial summaries and recent transactions. |
| **Income** | Money added to balance. |
| **Expense** | Money subtracted from balance. |
| **Flutter** | Framework for building cross-platform apps for mobile, web, desktop. |
