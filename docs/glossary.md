# Glossary

| Term | Definition |
|------|-------------|
| **CRUD** | Create, Read, Update, Delete operations for transactions. |
| **Admin** | User with elevated privileges to view registered users. |
| **Dashboard** | Main screen showing financial summary and recent transactions. |
| **Income** | Money added to balance. |
| **Expense** | Money subtracted from balance. |
| **React Native** | Framework for building cross-platform mobile apps. |
| **Expo** | Toolchain simplifying React Native development and deployment. |
