# External Interfaces

## Authentication Interface
- **Service:** Clerk  
- **Purpose:** Handles signup, login, and session validation.

## Database Interface
- **Service:** PostgreSQL (Neon)  
- **Purpose:** Stores user and transaction data.

## Frontend-Backend API Interface
- **Protocol:** REST (HTTPS)  
- **Format:** JSON  
- **Endpoints:** CRUD for transactions, summary fetch, and admin user view.
